# Tic-Tac-Toe #

A Tic-Tac-Toe game implemented in python that can interactively play against a human player and never lose.


## Usage ##

`python tic-tac-toe.py`


## How to contribute ##

1. Fork this repo
2. Fix, Modify, Extend and **Test**
3. Commit
4. Create a pull request
